URL Shortener

API

The API endpoints below are available. The number goes at the end of https://urlshortener.johnnyt001.repl.co/api/shorturl/ .

POST /api/shorturl

Create a shorter URL.

Request

To obtain a short URL make a POST request with a JSON payload. The url key must be included and its value set to the URL you want shortened. 

When you visit /api/shorturl/, you will be redirected to the shortened URL.
Response

The response includes both the original URL/ and the parameter. An Example is Below. 

{ original_url : 'https://freeCodeCamp.org', short_url : 1}
